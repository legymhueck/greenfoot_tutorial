Greenfoot Tutorial Workspace Presets

Open any file in this folder with VS Code to switch quickly to a chapter setup.

Each preset:
- uses Java 21 at /usr/lib/jvm/java-21-openjdk
- points Java source paths to one tutorial scenario
- includes the local greenfoot source folder for import resolution
